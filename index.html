📖 Manual de Usuario

🎯 ¿Qué es esta aplicación?

Transcriptor Médico Pro convierte archivos de audio en texto escrito usando inteligencia artificial (Whisper de OpenAI via Groq). Ideal para médicos, profesionales de salud, estudiantes y cualquier persona que necesite transcribir dictados de voz.

🔑 Configuración Inicial (API Key)

  • Necesitas una API Key gratuita de Groq para usar la app
  • Obtén tu key en console.groq.com/keys (registro gratis)
  • Pega tu key (empieza con gsk_) y presiona "Guardar Key"
  • La key se guarda localmente en tu navegador - solo la ingresas una vez
💡 Tip: Tu API Key nunca sale de tu dispositivo. Se almacena de forma segura en tu navegador.

📤 Subir Archivos de Audio

  • Arrastra y suelta archivos sobre la zona de carga
  • O toca/haz clic para seleccionar desde tu dispositivo
  • Formatos soportados: MP3, WAV, OGG, M4A, WebM, FLAC
  • Tamaño máximo: 25MB por archivo
  • Puedes subir múltiples archivos a la vez

🔊 Opciones de Mejora de Audio

Antes de transcribir, puedes activar estas opciones para mejorar la calidad:

  • 🔊 Normalizar: Ajusta automáticamente el volumen al nivel óptimo. Útil si el audio fue grabado muy bajo o muy alto.
  • 🔇 Reducir ruido: Aplica filtros para eliminar ruidos de fondo molestos como zumbidos, aire acondicionado, o siseo. Elimina frecuencias muy bajas (<80Hz) y muy altas (>12kHz).
  • 📦 Comprimir: Reduce el rango dinámico del audio. Esto significa que las partes muy suaves se hacen más audibles y las muy fuertes se suavizan. Ideal para: grabaciones con volumen irregular, voces que varían mucho entre susurros y gritos, o entrevistas donde algunas personas hablan más bajo que otras.
  • ➕ Unir audios: Si subes varios archivos y activas esta opción, el texto transcrito de todos los audios se unirá en una sola transcripción en el editor.
💡 Recomendación: Para grabaciones de celular o dictados casuales, activa las tres opciones. Para audio de buena calidad (micrófonos profesionales), puedes desactivarlas.

🎤 Transcribir

  • Presiona el botón "Transcribir" para iniciar
  • Los archivos se procesan uno por uno (secuencialmente)
  • Puedes ver el progreso en la barra y el archivo actual
  • Cada transcripción aparece en una pestaña separada
  • Puedes editar el texto mientras se procesan otros archivos

✏️ Editor de Texto

  • Negrita, cursiva, subrayado: Selecciona texto y usa los botones o atajos
  • Buscar y reemplazar: Botón 🔍 o Ctrl+H
  • Deshacer/Rehacer: Botones ↶ ↷ o Ctrl+Z / Ctrl+Y
  • Tamaño de fuente: Selector desplegable (14px - 24px)
  • Limpiar formato: Quita todo el formato del texto seleccionado
  • Pestañas: Si transcribes varios archivos, cada uno tiene su pestaña

📥 Descargar Transcripción

  • RTF (Word): Compatible con Microsoft Word, OpenOffice, LibreOffice. Mantiene formato básico.
  • PDF: Documento con formato profesional, ideal para compartir o imprimir.
  • TXT: Texto plano sin formato. Abre en cualquier editor.
  • HTML: Página web con estilos. Abre en cualquier navegador.

📋 Otras Funciones

  • Copiar: Copia todo el texto al portapapeles
  • Limpiar: Borra todos los archivos y transcripciones
  • Tema oscuro/claro: Botón 🌙/☀️ en la esquina superior derecha
  • Contador de palabras: Se actualiza en tiempo real mientras editas

🩺 Modo Profesional Pro

  • Activación: Toca el botón "Profesional" en la barra lateral.
  • Plantillas: Selecciona el tipo de estudio (ORL, Cardiología, Imágenes).
  • IA Estructuradora: La app usará inteligencia avanzada para organizar tu dictado en el formato médico correcto.
  • Unión Inteligente: Si unes audios en Modo Pro, el sistema generará un único reporte consolidado y estructurado.

📄 Configuración de Reportes (PDF)

  • Personalización: Botón 📋 en la toolbar para configurar encabezados, pies de página y datos de la clínica.
  • Datos del Paciente: Completa Nombre, DNI y Procedimiento para que aparezcan en el reporte final.
  • Justificación Profesional: El texto se alinea automáticamente para un aspecto editorial.
  • Firma: Se reserva un espacio automático al final a la derecha para tu sello y firma.

🎙️ Grabadora de Voz

  • Usa el botón "Grabar Audio" para dictar directamente desde tu micrófono.
  • Puedes pausar y reanudar o detener para generar la transcripción inmediata.
  • Ideal para dictados rápidos sin necesidad de subir archivos.

⌨️ Atajos de Teclado

  • Ctrl+B → Negrita
  • Ctrl+I → Cursiva
  • Ctrl+U → Subrayado
  • Ctrl+H → Buscar y reemplazar
  • Ctrl+Z → Deshacer
  • Ctrl+Y → Rehacer

📱 Uso en Celular

La aplicación está optimizada para dispositivos móviles:

  • La interfaz se adapta automáticamente a pantallas pequeñas
  • Puedes grabar audio con tu app de grabadora y subirlo directamente
  • Todos los formatos de descarga funcionan en móvil
  • Puedes agregar la web a tu pantalla de inicio como si fuera una app

❓ Solución de Problemas

  • "API 401": Tu API Key es inválida o expiró. Genera una nueva en Groq.
  • "API 413": El archivo es muy grande. Máximo 25MB.
  • "API 429": Demasiadas solicitudes. Espera un momento e intenta de nuevo.
  • Audio no se procesa: Verifica que el formato sea compatible. Intenta convertir a MP3.
  • Descarga no funciona: Verifica que tengas texto en el editor. Prueba otro formato.
